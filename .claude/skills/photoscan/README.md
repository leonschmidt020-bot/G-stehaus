# Design from Image Skill

Verwandle Screenshots und Design-Mockups in produktionsfertigen HTML/CSS/JS Code mit pixelgenauer Präzision.

## 🎯 Was macht dieses Skill?

Dieses Skill analysiert Bilder von Websites oder Designs und erstellt:
- ✅ Pixelgenauen HTML/CSS/JS Code
- ✅ Extrahierte Design-Tokens (Farben, Schriften, Spacing)
- ✅ Responsive Layouts
- ✅ Moderne Best Practices
- ✅ Accessibility-Features

## 🚀 Verwendung

### Beispiel 1: Screenshot hochladen

## 💡 Tipps für beste Ergebnisse

### Screenshot-Qualität
✅ **Gut:**
- Hohe Auflösung (mindestens 1280px breit)
- Klare, scharfe Bilder
- Vollständige Sections sichtbar
- Desktop-Ansicht (für primäres Design)

❌ **Vermeiden:**
- Verschwommene Bilder
- Zu kleine Screenshots
- Stark komprimierte Bilder
- Teilweise abgeschnittene Elemente

### Was du angeben solltest
- "Erstelle diese Landing Page nach"
- "Achte besonders auf [Farben/Spacing/Animationen]"
- "Mobile-First Design"
- "Füge moderne Hover-Effekte hinzu"

## 🎓 Best Practices

### Farben extrahieren
- Verwende Color Picker Tools
- Runde auf web-sichere Farben
- Dokumentiere alle Farb-Varianten

### Spacing messen
- Verwende 8px-Grid als Basis
- Messe relative Abstände
- Erstelle konsistentes System

### Fonts identifizieren
- Erkenne Serif vs. Sans-serif
- Schätze Größen relativ
- Verwende Web-Font Fallbacks

### Layout nachbauen
- Zähle Grid-Columns genau
- Beachte Alignment
- Preserve White Space
